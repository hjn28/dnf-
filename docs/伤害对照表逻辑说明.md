# 伤害对照表 — 逻辑说明

## 精确指令（下次直接引用）

```
有四组数据（三年、淘气、起源、老王），按照c24个大值，奶24个大值；
最大的c使用最小的奶（大c配小奶）；
然后计算出来的结果，进行排序；
排完序之后，计算均值、方差；
在均值之上的，优先填充混子和小c（从大到小处理）；
在均值之下的，优先填充大c和太阳奶（从小到大处理）；
计算完之后，看均值和方差；
然后进行奶、c之间的交换（同列不跨列），
达到最后均值基本都在附近。
```

## 执行流程

```
① 取24个最大c（降序） + 24个最大奶（配对时反转取最小端）
② 大c配小奶：c[i]↔奶[i]
③ 基伤 = c × rate(奶) ÷ 1000
④ 排序、计算均值
⑤ 填充：
   低于均值 → 大c优先 → 太阳奶(×1.24)
   高于均值 → 混子优先 → 小c
⑥ 兜底补全
⑦ 同列交换（c/奶）→ 方差收敛
⑧ 最终伤害
```

## 结果

```
基伤: 均值=4206
填充后最低值 ≈ 3668
其余22行极差 ≈ 4000~4500
最高2行: c=7851/5001 配小奶后仍偏高（天然差距）
```
